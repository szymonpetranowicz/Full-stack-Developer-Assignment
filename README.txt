Requirements for running this project.

1. Install Apache and MySQL servers locally, I'll recommend you to download Xampp.
2. Download all files from the repository.
3. Run Apache and MySQL servers
4. Type in url 'localhost/phpmyadmin/'
5. Create new database and name it 'simple_table'
6. In created database, import database from file 'simple.sql'
7. Move downloaded files to your local htdocs. (If you downloaded Xampp, it will be setted in your installed files xampp/htdocs)
8. Type in url "localhost/basse.php".
